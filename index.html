  • Teleprompter

    Teleprompter Project

    This project is a simple web-based teleprompter created for demonstrative purposes. It showcases the basic functionality of a teleprompter using HTML, CSS, and JavaScript.

    Features
    1. It has option to change the text font size to accordingly make it easy to read from a distance.
    2. It can scroll the text at +ve as well as -ve speed that can be adjusted using the speed control bar.
    3. It provides controls to start, pause, and toggle fullscreen .
    4. It allows the user to input their own text as well as selecting a text file to be displayed on the teleprompter.
